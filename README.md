# BYR-template 北邮人简介内容模板
直接将html代码复制进编辑器源码模式即可使用;

- template是动漫模板
- template2是剧集/电影模板

v1版本是当前使用的，未针对非Windows用户做字体优化

v2版本使用自定义css，提供全平台字体支持

v2内建字体样式：
